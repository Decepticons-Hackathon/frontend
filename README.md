## Запуск

1. Клонируем репозиторий

2. Устанавливаем необходимые зависимости

```gitbash
npm ci
```

3. Запускаем скрипт сборки приложения

```gitbash
npm run build
```

4. Запускаем приложение на локальном сервере в прод-режиме

```gitbash
npm run start
```
Либо в дев-режиме, без сборки
```gitbash
npm run dev
```
